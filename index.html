Recent Project Work

Generating Dog Images: Python, Tensorflow, Keras

A recent Kaggle competition made by the Kaggle team to use generative methods to create pictures of dogs that have never been taken by a camera before. Generative adversarial networks, introduced by Ian Goodfellow, allow this by having a neural network called the generator that takes a random uniformly distributed number vector as input then outputing an image which is taken into another neural network called the discriminator along with real pictures from training data and tries to distinguish them as real and fake. There are more details to this where I will attempt to explain as much as possible in my Kaggle kernal which is still a work in progress as there is month to go in the competition.
